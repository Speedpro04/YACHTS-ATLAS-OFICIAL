# Yachts Atlas — Design System Padrão Ouro 🏆

Este guia define os padrões visuais para manter a estética High-Ticket em todo o ecossistema Yachts Atlas.

## 1. Paleta de Cores (Cores Reais)
Usamos tons que transmitem luxo, autoridade e solidez.

*   **Deep Navy (Fundo):** `#020617` — O azul mais profundo, transmitindo seriedade.
*   **Rich Gold (Primária):** `#C5A059` — O tom de ouro para botões e CTAs principais.
*   **Champagne (#F1E4C3):** Usado para textos secundários e detalhes finos.
*   **Status do Ativo:**
    *   **Gold (Padrão Ouro):** `#C5A059`
    *   **Silver (Prata):** `#94A3B8` (Slate Silver)
    *   **Bronze:** `#B45309` (Amber Intenso)

## 2. Tipografia
*   **Fonte Principal:** **"Outfit"** (Google Fonts).
    *   *Por que:* É uma fonte geométrica com um toque moderno e "limpo", usada por marcas de tecnologia de luxo.
*   **Fonte de Apoio:** **"Inter"**.
    *   *Por que:* Excelente legibilidade para dados técnicos e tabelas.

## 3. Estética UI (O Toque de Luxo)
*   **Glassmorphism:** Uso de fundos levemente transparentes com desfoque (blur) para criar profundidade.
*   **Gradients Metálicos:** Sempre que possível, usar gradientes sutis do `gold-500` para o `gold-300` para simular o brilho do metal.
*   **Bordas Finas:** Usar bordas de `0.5px` ou `1px` com cores suaves (`navy-700`) para separar seções sem poluir o visual.

## 4. Ícones
*   **Biblioteca:** Lucide-React.
*   **Estilo:** Traços finos e elegantes. Nunca usar ícones muito "coloridos" ou infantis.
