# Yachts Atlas — Argumento Estratégico

## Positioning: Por Que Este Produto É Único

---

### O Mercado

O mercado náutico de luxo movimenta billions anualmente. Iates de 10m+ valem de R$2MM a R$50MM+. Jet-skis de alta performance custam R$150K+.

Este público nãoquer aplicativos plastikos. Querem:
- **Prestígio** — Interface que reflita o valor do ativo
- **Integridade** — Prova de que tudo está documentado
- **Valor real** — Documentação que aumento o preço de venda
- **Indicação** — Seguradoras que CONFIELEM no produto

---

### O Diferencial: Classificação Do Ativo

O Yachts Atlas não é só um repositório. É um **sistema de certificação**.

| Classificação | Descrição | Requisitos |
|---|---|---|
| **Bronze** | Ativo em conformidade básica | Documentos obrigatórios (RGP, seguro) |
| **Silver** | Ativo bem documentado | Bronze + histórico de manutenções + fotos |
| **Gold** | Ativo de excelência | Silver + relatórios de survey atualizados + dossier completo |

**Barra de Progresso:**
O dono vê exatamente o que falta para atingir o próximo nível.

---

### O Dossier: Produto De Valor Real

3 níveis de dossier para negociação:

| Dossier | Para Quem | O Que Inclui |
|---|---|---|
| **Básico** | Venda direta | Registro, seguro, documento único |
| **Completo** | Corretor/banco | Básico + histórico de manutenções + fotos |
| **Premium** | Due diligence | Completo + relatórios de survey + certidões atualizadas +Hash de integridade |

### A Certificação: Yachts Atlas Certified
